## Discovery decisions
- Working mode: Coaching path.
- Visual mood: dark cockpit/broadcast feel for the live experience, clean editorial/data-journalism feel for calendar/standings/profiles, F1 constructor colors as the accent system throughout (not one fixed brand color).
- Dark mode scope: whole app dark by default (single consistent dark identity). Light mode treated as optional/post-POC, not a blocking requirement for this pass — no FR currently covers a light/dark toggle.
- Color coding: broadcast-standard, color-only (matches real F1 graphics: purple/green/yellow/white mini-sectors, red/yellow/white/green/blue tyre compounds). Explicit accessibility trade-off — no colorblind-safe secondary cue for the POC. Documented here as a known gap in case revisited later.

## Visual direction selected: Dark Editorial
Source: `.working/direction-dark-editorial.html` (one of 3 rendered directions: Cockpit HUD, Dark Editorial, Carbon Glass).
Rationale (from renderer): premium motorsport-publication feel rather than a pitwall HUD — calm dark background, generous whitespace, soft-rounded cards, humanist sans typeface, team colour restrained to small dots/badges/thin borders rather than backgrounds. Reading-friendly for a fan sitting through a 2-hour race.

## Visual direction refined: Dark Editorial shell + Cockpit HUD gap-list density (CONFIRMED, final)
Source: `.working/direction-dark-editorial-hybrid.html`.
Bohdan liked Cockpit HUD's compact single-line gap-list table (all drivers visible without scrolling) but preferred Dark Editorial's calmer shell/colors/typography for everything else. Final rule for DESIGN.md/EXPERIENCE.md distillation:
- Everything (topbar/nav, track map, lap chart, sector board, event timeline, cards, typography, color tokens) = Dark Editorial as captured above.
- Live Gap List table specifically = Dark Editorial's color tokens and font family, but Cockpit HUD's row density: tight padding (~6-8px vertical), ~12-12.5px font, single line per driver (Position | Driver code + small team-color dot | Gap with restrained `--sector-yellow` battle highlight (not HUD's harsh amber box) | Tyre dot + stint laps inline | Provisional championship-delta note inline) — no two-line driver+team-name stacking, no wrapping. Goal: full ~20-driver grid visible without scrolling.
- This dense-table pattern applies only to the Live Gap List. No other component borrows Cockpit HUD's monospace font, sharp corners, or glow effects.
